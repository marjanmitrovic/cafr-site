# Faza: Seminari i prijave

Dodato:
- admin kreira i objavljuje seminar
- član vidi seminare u dashboardu
- odobren član se prijavljuje i odjavljuje
- kontrola kapaciteta
- admin vidi spisak prijavljenih
- obaveštenje članu nakon prijave

Posle raspakivanja pokrenuti: npm install, npm run db:generate, npm run db:push, npm run db:seed, npm run dev.
